Ask a question and get an AI-powered answer grounded in the Claude Code expertise database.

Uses vector search + taxonomy + resource lookup to build context, then generates a cited answer.

Usage: /ask <question>

Examples:
- /ask what is the best stack for using claude code for product development?
- /ask how do agent swarms coordinate work?
- /ask what are the patterns for handling large codebases?

```bash
cd $WORKSPACE_DIR && python scripts/ask.py "$ARGUMENTS"
```
